# IterLogs Analyzer

Анализатор серверных метрик и логов, построенный исключительно на генераторах и функциях модуля `itertools`. Проект демонстрирует функциональный подход к обработке данных в Python, минимизируя расход оперативной памяти при работе с большими последовательностями.

## Используемые функции `itertools`

В проекте задействовано 13 методов библиотеки:

* **`product`**: Формирование матрицы возможных маршрутов (Cartesian product).
* **`cycle`**: Эмуляция алгоритма балансировки нагрузки Round-Robin.
* **`count`**: Бесконечный генератор уникальных ID транзакций.
* **`islice`**: Постраничная выборка и лимитирование бесконечных потоков.
* **`tee`**: Клонирование потока данных для независимой параллельной обработки метрик.
* **`filterfalse`**: Инвертированная фильтрация для выявления серверных ошибок (отсеивание статусов < 400).
* **`groupby`**: Агрегация и кластеризация логов по запрашиваемым URL.
* **`starmap`**: Векторное сложение входящего и исходящего трафика.
* **`accumulate`**: Расчет кумулятивного графика сетевой нагрузки.
* **`dropwhile`**: Пропуск стандартных метрик для поиска начала аномалий (задержек).
* **`takewhile`**: Изоляция диапазона метрик по заданному условию.
* **`chain`**: Склеивание разрозненных дампов логов в единый поток.
